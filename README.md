# Gantt 2025 08 バックアップ

このバージョンはGASの「GAS参照用ガントチャート」にひもづいていた「ガントチャートテンプレート」ライブラリの2026年08月時点でのスナップショットです。

（おそらく一年以上前からですが、遅くとも）この時点では「ガントチャート」以外の機能も登録されているので注意です。

個別の生徒の「スピードプランナー」というスプレッドシートに紐づいているスクリプトの実装を共通化するために使用されていました。

1. 月末月初のデータ移行処理をする「月末月初マクロ」

2. 毎週の学習計画を自動で生成し、サジェストする機能「週間計画作成マクロ」

3. 面談前に生徒に渡すプランナーシートに書かれているヒアリング項目の生成をする「ヒアリング項目作成」

4. (廃止済み)生徒の学習進捗を生徒が簡単に記せるようにするための「Googleフォーム作成」

5. その他移行用スクリプトなどレガシー

が定義されています。
