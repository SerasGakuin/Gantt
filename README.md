# ガントチャートテンプレート

このバージョンはGASの「GAS参照用ガントチャート」にひもづいていた「ガントチャートテンプレート」ライブラリのスナップショットです。

「ガントチャート」以外の機能も登録されているので注意です。

個別の生徒の「スピードプランナー」というスプレッドシートに紐づいているスクリプトの実装を共通化するために使用されていました。

2026-04-02 現在、

1. 呼び出し用のインターフェース
2. 月末月初のデータ移行処理をする「月末月初マクロ」
3. 毎週の学習計画を自動で生成し、サジェストする機能「週間計画作成マクロ」
4. 面談前に生徒に渡すプランナーシートに書かれているヒアリング項目の生成をする「ヒアリング項目作成」
5. 月末月初以外のタイミングでも教材のレパートリを変更できるようにするための「教材中途変更マクロ」
6. シートの数式を直す「シート構成修復マクロ」

これらの機能が実装されています。
